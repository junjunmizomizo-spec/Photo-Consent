# Harmony Design Guide

> Photo Consent Version 2.0 “Bloom”  
> “Harmony” is the private project name shared by the creators.

## Logo
Use the full approved logo for README and documentation. Use the square camera mark for PWA icons, Apple Touch Icon, and favicon. The splash presentation contains only the logo and the app name.

## Typography
- App title: 32px / 700
- Screen heading: 24px / 700
- Body: 16px / 400
- Helper text: 14px / 400

## Spacing
Use 8, 16, 24, 32, and 48px whenever practical.

## Radius
- Small: 12px
- Medium: 18px
- Large: 24px

## Motion
Motion should feel calm, short, and supportive. Respect `prefers-reduced-motion`.

## Writing tone
Gentle, direct, non-judgmental, and easy to understand.

The message “どの選択肢を選んでも大丈夫です。” is a core product principle.


## Version 2.1 UI refresh

### UI logo
The application interface uses a camera outline, circular lens, and leaf sprig. It does not use a heart. The stroke uses a soft teal-to-leaf gradient.

### Consent-choice colors
The five choices use a gradual spectrum without purple:

1. Green
2. Yellow
3. Orange
4. Light red
5. Red

The colors support scanning only. They must not imply that one participant choice is morally better than another.
