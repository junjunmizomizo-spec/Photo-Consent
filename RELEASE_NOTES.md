# Photo Consent Version 2.4.0 Official

リリース日：2026年7月21日

## 正式版に含まれる内容

### UI Refresh
- カメラ＋葉っぱのハートなしロゴ
- 緑から黄緑へのブランドグラデーション
- Welcome画面の中央配置
- PCとiPhoneに対応したレスポンシブ表示

### Participant Flow
- 名前入力、撮影希望、確認、完了画面のデザイン統一
- 5つの選択肢を緑から赤へ段階的に色分け
- 保存完了後の3秒自動リセット
- 「次の方へ」ボタン

### Admin Dashboard
- 管理者ログイン画面
- イベント概要
- イベント管理
- 回答一覧
- 回答の検索、編集、削除
- CSV出力
- JSONバックアップと復元

### Quality and Accessibility
- 未入力時の画面内メッセージ
- 入力状態に応じたボタンの有効化
- Enterキー操作
- 画面移動時のフォーカス調整
- キーボードフォーカス表示
- `prefers-reduced-motion` 対応
- 強制カラーモードへの配慮

## データについて

回答とイベント情報はブラウザのローカルストレージへ保存されます。端末故障、ブラウザデータ削除、別端末への移行に備え、管理画面から定期的にバックアップしてください。
