# Photo Consent Version 1.4.1

iPhoneで、フェードの最後だけ急に表示されるように見える問題を調整した版です。

## 調整内容

- フェードインを300msから480msへ延長
- 透明度を4段階で少しずつ変化
- 横移動をやめ、下からわずかに浮かぶ動きへ変更
- 終了直前まで透明度がなめらかに変化するよう調整
- Safariでアニメーション終了時に起きる小さな「パッ」を防止
- フェードアウトも少し柔らかく調整

## 更新方法

ZIPを解凍し、中の8ファイルをGitHub上の既存ファイルへ上書きしてください。

iPhoneではSafariを再読み込みしてください。ホーム画面版で古い動作が残る場合は、一度削除して追加し直してください。
