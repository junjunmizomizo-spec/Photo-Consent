# Photo Consent Version 1.3 — Sprint 4-3

「Harmony」は制作者2人だけで共有するプロジェクト名です。公開画面には表示しません。

## Sprint 4-3 の変更

- 参加者向け画面は、緑と白を中心にした安心感のある雰囲気を維持
- 管理者向け画面は、青みのある業務向け配色へ変更
- 管理画面に、現在のイベント名・回答数・状態をまとめたカードを追加
- トースト表示時間を約4秒へ延長
- 通知をタップすると閉じられる
- CSVとバックアップは、iPhoneの保存画面から戻った後にも通知
- CSVは `📄`、バックアップは `💾` で内容を判別
