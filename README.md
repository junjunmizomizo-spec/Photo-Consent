# Photo Consent Version 1.5 — Stable Motion

「Harmony」は制作者2人だけで共有するプロジェクト名です。公開画面には表示しません。

## 今回の方針

iPhoneでの安定性を最優先し、画面全体を重ねる方式は採用していません。

Version 1.4.2までの安定したレイアウトへ戻し、表示されたカードだけを軽くフェードさせます。

## 変更内容

- 安定した `display:none` / `display:block` の画面切り替えを維持
- 画面全体のクロスフェードを廃止
- 表示されたカードだけを約260msで自然に表示
- 高さ崩れや画面消失を防止
- ボタンと選択カードの押し込み感を追加
- iPhone Safariでのちらつきやカクつきを抑制
- 「動きを減らす」設定に対応
- イベント管理、CSV、バックアップ、トースト機能を維持

## 確認方法

1. 「はじめる」を押す
2. 「戻る」を押す
3. 名前入力から選択画面へ進む
4. 確認画面まで進む

画面は確実に切り替わり、カードだけが軽く柔らかく表示されれば正常です。
