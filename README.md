# Photo Consent Version 1.3 — Sprint 4-1

「Harmony」は制作者2人だけで共有するプロジェクト名です。公開画面には表示しません。

## Sprint 4-1 の変更

最初の画面を、より安心して選べるデザインへ整えました。

- ウェルカム画面のレイアウトを全面調整
- 「どの選択肢を選んでも大丈夫です。」を分かりやすく表示
- 「今日は撮られたくない」という気持ちも大切にする文言を追加
- 説明文を読みやすいカードに整理
- iPhoneで押しやすい横幅いっぱいの「はじめる」ボタン
- 余白、文字サイズ、背景装飾を調整
- 公開画面の名称は引き続き「Photo Consent」のみ

## 今回は変更していない機能

Version 1.2までの次の機能は、そのまま利用できます。

- イベント管理
- イベント終了
- CSV出力
- 回答の編集・削除
- バックアップ・復元
- PWA・オフライン対応

## GitHub Pagesへの更新

1. ZIPを解凍します。
2. 中の8ファイルをGitHub上の既存ファイルへ上書きします。
3. GitHub Pagesを開き直して確認します。

以前の画面が表示される場合は、Safariを再読み込みしてください。
