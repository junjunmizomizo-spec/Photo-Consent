# Photo Consent Version 1.4.2

iPhoneでフェードの最後に一瞬カクつく・ちらつく問題を修正した版です。

## 修正内容

- フェード終了時にアニメーションを即座に解除しない方式へ変更
- iPhone Safariで起きていた1フレームのちらつきを防止
- 画面遷移は透明度だけで行い、変形処理を外して描画を安定化
- フェードインは約460ms
- フェードアウトは約170ms
- 次の遷移が始まるまで、完了状態を維持
- 「動きを減らす」設定では即時切り替え

## 更新方法

ZIPを解凍し、中の8ファイルをGitHub上の既存ファイルへ上書きしてください。

iPhoneではSafariを再読み込みしてください。ホーム画面版で古い動作が残る場合は、一度削除して追加し直してください。
