# Photo Consent Version 1.1 — Sprint 3-2

「Harmony」は制作者2人だけで共有するプロジェクト名です。公開画面には表示しません。

## 今回の追加

- イベント終了ウィザード
- 終了時のCSV保存確認
- 回答を残す／削除する選択
- 終了後に新しいイベントを作成
- CSVにイベント名、作成日、回答数を追加
- CSVファイル名を `日付_イベント名.csv` に変更
- イベント削除時にイベント名の再入力を要求
- イベント削除ボタンを赤色で表示
- イベント一覧に回答件数を表示

## イベント終了の流れ

1. 「イベントを終了する」を押します。
2. CSVを書き出すか選びます。
3. 回答を残すか削除するか選びます。
4. 新しいイベントを作成するか選びます。

## 管理画面

画面右上の歯車を約2秒長押しします。初期パスコードは `1234` です。

## GitHub Pagesへの更新

ZIPを解凍し、中の8ファイルをGitHub上の既存ファイルへ上書きしてください。

回答データはブラウザのLocalStorageに保存されているため、通常はコード更新後も維持されます。
ブラウザデータを削除すると回答も消えるため、必要なデータはCSVで保存してください。
