# Photo Consent Version 1.4

「Harmony」は制作者2人だけで共有するプロジェクト名です。公開画面には表示しません。

## Version 1.4 のテーマ

iPhone Safariでの画面遷移を最優先に調整しました。

## 変更内容

- CSSだけに依存した画面遷移を廃止
- JavaScriptのWeb Animations APIで画面を切り替え
- 現在の画面を短くフェードアウト
- 次の画面をゆっくりフェードイン
- iPhone Safariが描画を省略しないよう、表示後に2フレーム待ってからアニメーション
- GPU描画を使いやすくする `translate3d` を採用
- 連続タップによる二重画面遷移を防止
- 端末の「動きを減らす」設定では即時切り替え
- Version 1.3までのイベント管理、CSV、バックアップ、トースト機能を維持

## GitHub Pagesへの更新

1. ZIPを解凍します。
2. 中の8ファイルをGitHub上の既存ファイルへ上書きします。
3. iPhoneのSafariで公開ページを再読み込みします。
4. ホーム画面から使っている場合は、古いアイコンを一度削除して追加し直すと確実です。

## 確認ポイント

参加者画面で次の順に確認してください。

1. 「はじめる」
2. 名前入力後の「次へ」
3. 選択後の「確認する」
4. 「送信する」

各画面が、一度わずかに消えてから次の画面が柔らかく現れれば正常です。
