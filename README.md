# Photo Consent Version 1.5 — Sprint 5-1

「Harmony」は制作者2人だけで共有するプロジェクト名です。公開画面には表示しません。

## Sprint 5-1 の目的

iPhoneで安定する新しい画面遷移方式を、最初の2画面だけで実機テストします。

## 今回変更した画面

- ウェルカム画面
- 名前入力画面

この2画面だけを同じ位置へ重ね、`display:none` を使わずに透明度で切り替えます。

## 新しい遷移方式

- 2画面を最初から同じ場所に配置
- `opacity`
- `visibility`
- `pointer-events`

だけで切り替えます。

画面の入れ替えではなく、片方が薄くなりながら、もう片方が濃くなるクロスフェードです。

## 実機での確認

1. 最初の画面で「はじめる」を押す
2. 名前入力画面で「戻る」を押す
3. 何度か往復する

最後にちらつかず、2つの画面が自然に入れ替われば成功です。

## 今回変更していない部分

- 名前入力以降の画面遷移
- イベント管理
- CSV
- バックアップ
- トースト通知
- 管理画面

これらはVersion 1.4.2までの仕組みを維持しています。
