# Photo Consent

写真撮影について、一人ひとりの希望を確認・管理するためのWebアプリです。

「Harmony」は制作者2人だけで共有するプロジェクト名です。
公開画面や利用者向けのアプリ名には表示しません。

## 主な機能

- 名前入力と5段階の撮影希望
- 確認・修正・送信
- 3秒後に最初の画面へ自動復帰
- イベントごとの回答管理
- 回答一覧・検索・編集・削除
- CSV出力
- PWA・オフライン対応
- GitHub Pages対応

## 管理画面

画面右上の薄い歯車を約2秒長押しします。

初期パスコードは `1234` です。
変更する場合は `script.js` 内の `P="1234"` を変更してください。

## GitHub Pagesへの公開

ZIPを解凍し、中のファイルをすべてGitHubリポジトリのルートへアップロードします。
その後、`Settings` → `Pages` で `Deploy from a branch`、`main`、`/(root)` を選びます。

## 保存について

回答は利用中のブラウザのLocalStorageに保存されます。
ブラウザのデータを削除すると回答も消えるため、イベント終了後はCSVを書き出してください。
