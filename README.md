# Photo Consent Version 1.2

「Harmony」は制作者2人だけで共有するプロジェクト名です。公開画面には表示しません。

## Version 1.2 の追加

- 全イベント・全回答のJSONバックアップ
- バックアップファイルからの復元
- 不正なバックアップファイルの拒否
- 復元前の上書き確認
- バックアップファイル名に日付を追加
- イベントの「進行中」「終了済み」表示
- 終了済みイベントの再開
- イベント一覧に回答数と状態を表示
- イベントの作成日・最終更新日時を表示

## バックアップ

管理画面の「バックアップを保存」を押すと、次の形式で保存されます。

```text
2026-07-20_PhotoConsent_backup.json
```

このファイルには、すべてのイベントと回答が含まれます。

## 復元

「バックアップから復元」を押し、保存済みのJSONファイルを選びます。

復元すると現在のデータは上書きされます。必要であれば、復元前にもバックアップを保存してください。
