# Photo Consent v1.1.0 — Sprint 3-1

「Harmony」は制作者2人だけで共有するプロジェクト名です。利用者向け画面には表示されません。

## Sprint 3-1で追加・改善した機能

- 管理画面の最上部に「現在のイベント」を大きく表示
- 現在のイベントの回答総数を表示
- 新しいイベントを作成
- 作成後、そのイベントへ自動で切り替え
- プルダウンからイベントを切り替え
- イベント一覧に各イベントの回答数を表示
- 同じ名前のイベントの重複作成を防止
- イベント切り替え時に検索・絞り込みをリセット

## 管理画面

画面右上の薄い歯車を約2秒長押しします。初期パスコードは `1234` です。

## GitHubへの反映

1. ZIPを解凍します。
2. GitHub上の既存ファイルを、解凍したファイルで上書きします。
3. 反映後、Safariでページを再読み込みします。
4. 古い画面が残る場合は、Safariのタブを閉じてから再度開きます。

回答はブラウザのLocalStorageに保存されるため、通常のファイル上書きでは既存回答は消えません。
