# Photo Consent Version 1.3 — Sprint 4-2

「Harmony」は制作者2人だけで共有するプロジェクト名です。公開画面には表示しません。

## Sprint 4-2 の変更

操作の流れを邪魔しない、控えめで心地よい動きを追加しました。

- 画面切り替え時のフェード＋スライド
- 選択カードを押したときの軽い縮小アニメーション
- 選択確定時の小さなポップ表現
- 完了画面のチェックマークを自然に表示
- CSV書き出し後のトースト通知
- バックアップ保存後のトースト通知
- バックアップ復元後のトースト通知
- 不正なバックアップファイルのエラー通知
- 「動きを減らす」端末設定に対応

## トースト通知

保存や復元の完了時に、画面下部へ約2秒だけ通知が表示されます。

例：

- `CSVを書き出しました。`
- `バックアップを保存しました。`
- `バックアップを復元しました。`
