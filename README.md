# Photo Consent

写真撮影について、一人ひとりの気持ちを大切に確認するためのWebアプリです。

## Version 2.1 “UI Refresh”

- カメラ＋葉っぱのシンプルなロゴ（ハートなし）
- Welcome画面を中央配置のブランドデザインへ刷新
- 選択肢は顔アイコンを使わず、番号と色で表示
- 色は緑 → 黄色 → オレンジ → 薄い赤 → 赤
- PC・iPhone・PWAで共通のレイアウト
- イベント管理、CSV、バックアップ機能を維持

## 選択肢

1. 写真OK
2. 顔が写っていなければOK
3. みんなと写っていればOK
4. ボケていたり、身体の一部分ならOK
5. 今回は撮影を見送り
